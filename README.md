# Bun Full Stack Starter

This project demonstrates:

- TSX based static components rendered server-side
- HTML streaming via `react-dom/server` in Bun

## Commands

```bash
bun install
bun run dev
```

Then visit http://localhost:3000

## Commits

Easy commits via gitmoji

```bash
git add --all
git commit
```
